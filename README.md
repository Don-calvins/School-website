# Jaramogi Oginga Odinga University of Science and Technology (JOOUST) Website

A modern, responsive university website built with HTML5, Tailwind CSS, and vanilla JavaScript.

## Features

- **Responsive Design**: Fully mobile-friendly with adaptive layouts
- **Modern UI**: Professional color scheme with smooth animations
- **Navigation**: Sticky navigation bar with mobile hamburger menu
- **Announcement Banner**: Marquee-style scrolling announcements
- **Multi-page Layout**: Dedicated pages for different university sections
- **Social Integration**: Social media links in header
- **Student Portal Access**: Quick link to student portal
- **Font Awesome Icons**: Professional icon library

## Pages

- **index.html** - Main homepage with marquee announcements, header, navigation, and hero section
- **Academics.html** - Academic programs and departments page
- **Admissions.html** - Admissions information and application process (coming soon)
- **Research.html** - Research initiatives and opportunities (coming soon)
- **Students.html** - Student resources and services (coming soon)
- **Contact.html** - Contact information and inquiry form (coming soon)

## Tech Stack

- **Frontend**: HTML5, CSS3, Tailwind CSS
- **Icons**: Font Awesome 6.4.0
- **Styling**: Responsive Tailwind CSS framework
- **JavaScript**: Vanilla JS for interactive features

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/Don-calvins/School-website.git
   cd School-website
   ```

2. Open in your browser:
   - Simply open `index.html` in your web browser
   - Or use a local server: `python -m http.server 8000`
